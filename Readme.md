# Pixel Quest (가제)

socket.io와 canvas를 이용한 간단한(?) 게임을 제작.

###

주요 기술

- HTML canvas
- socket.io
- typescript

# How to use

#### server/

```jsx
// localhost server on
npm run dev
or
yarn dev
```

#### frontend/

```jsx
// typescript 빌드 (with watch)
npm run build
or
yarn build
```

```jsx
// localhost server on
npm run start
or
yarn start
```
